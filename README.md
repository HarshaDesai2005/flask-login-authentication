# Login Authentication System

A simple login authentication system built with HTML/CSS frontend, Python Flask backend, and MySQL database.

## Setup Instructions

1. Install Python dependencies:
```bash
pip install -r requirements.txt
```

2. Set up MySQL database:
- Create a database named 'login_auth'
- Update the database configuration in `config.py`

3. Run the application:
```bash
python app.py
```

4. Access the application at `http://localhost:5000`

## Features
- User registration
- User login
- Password hashing
- Session management
- Responsive design
